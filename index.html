You can make own facebook gallery on big-screen. It's built up on backbone.tv. This project can be run on both yeoman 0.9.6 express-stack or foreman that is tools for heroku. You can choose either to develop the project. Here is fallery demo. It's gonna be show how to backbone.tv is works.

backbone.tv

backbone.tv is the backbone extension library to easily make a webapp in tv style. It was composed by several modules. One is backbone.keyevent that is help to handle key event simply, in View-Hierarchy-Structure. Task, Resource Manager and XHR Web-Worker is gonna give to you enhanced performance. Also supports Web-Animation internally
